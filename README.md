Stars Invaders 
Genre : Space Invaders / Arcade  
Plateforme : Windows (SFML 3.0 ou supérieur)  
Auteur : Lino Lomboni 
Pitch narratif 
Le joueur incarne un résistant de l’univers de Star Wars et doit sauver la galaxie, rien que ça !? Il sera confronté à des 
vagues d’ennemis de plus en plus puissants et résistants. Son but est d’arriver au bout de la dernière vague pour 
battre son ennemi juré et libérer cette bonne vielle galaxie des griffes de l’Empire 
Installation 
Prérequis :  
• Windows 10 ou supérieur  
• SFML 3.0.x installé  
• Compilateur C++ compatible C++17 (Visual Studio recommandé)  
Configuration des assets : 
Les fichiers fonts, textures et autres assets doivent se trouver dans le dossier assets à la racine du projet. Exemple :  
assets/ fonts/ Arial.ttf  
Starjedi.otf 
textures/ enemy.png  
enemy2.png  
player.png  
boss.png  
Compilation : 
• Ouvrir le projet dans Visual Studio  
• Ajouter les dossiers SFML dans les include/lib  
• Compiler et exécuter  
Commandes 
Déplacement Haut-Bas-Gauche-Droite avec les flèches directionnelles.  
Durée d’un run  
Une partie peut durer de 2min30 si vous êtes vraiment bon à 4min30 environ pour les plus débutants 
Notes: Arriver à 0pv vous ramène au tout début du jeu, Attention pas de checkpoint, ni de buff pour récupérer de la 
vie. Vous êtes seul sur le champ de bataille seul votre talent vous aidera à abattre l’ennemi.
